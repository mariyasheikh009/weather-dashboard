# **WEATHER DASHBOARD 🌦️**

---

## ✨ Overview

Weather Dashboard is a simple and user-friendly web application that allows users to search for any city and view real-time weather information such as temperature, humidity, and weather conditions.
It is designed with a clean interface and smooth user experience.

---

## 🚀 Features

* 🔍 Search weather by city
* 🌡️ Real-time temperature updates
* 💧 Humidity & weather conditions
* 🎨 Clean and responsive interface
